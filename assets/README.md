# Замена изображений

Если вы хотите использовать свои собственные изображения для приложения:

## Шаг 1: Подготовьте изображения

Вам нужно три изображения:
- `background.jpg` - фоновое изображение для GUI (рекомендуемый размер: 800x600)
- `icon.jpg` - иконка для системного трея (рекомендуемый размер: 256x256)
- `app_icon.jpg` - иконка окна приложения (рекомендуемый размер: 64x64)

## Шаг 2: Замените файлы

Скопируйте ваши изображения в папку `assets/`, заменив существующие файлы:
```
assets/
├── background.jpg   # Замените этот файл
├── icon.jpg         # Замените этот файл
└── app_icon.jpg     # Замените этот файл
```

## Шаг 3: Пересоберите приложение (если требуется)

Если вы создаете EXE файл с помощью PyInstaller:
```bash
pyinstaller DiscordRPCAppChanger.spec
```

Все изображения будут автоматически включены в сборку.

## Примечание

- Используйте формат JPG для совместимости
- Убедитесь, что имена файлов точно совпадают
- Изображения включены в репозиторий, поэтому другие пользователи получат их автоматически
